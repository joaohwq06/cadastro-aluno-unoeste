# Projeto Cadastro de Aluno

Este projeto foi desenvolvido para a atividade da disciplina de Desenvolvimento Web.

O objetivo do trabalho foi criar uma aplicação web utilizando Node.js e Express que fosse capaz de receber e processar os dados enviados por um formulário HTML.

## O que o projeto faz

- Exibe um formulário de cadastro de aluno
- Recebe os dados (nome, idade e curso)
- Mostra na tela as informações cadastradas
- Possui estilização básica utilizando CSS

## Tecnologias utilizadas

- Node.js
- Express
- HTML
- CSS

## Como executar

1. Instalar as dependências:
   npm install

2. Rodar o servidor:
   node server.js

3. Acessar no navegador:
   http://localhost:3000
